## Phase 1: 코어 전투 (2주)

### 목표

싱글 전투 플레이 가능

### 포함 기능

| 기능 | 상세 | 우선순위 |
|------|------|----------|
| 스탯 시스템 | 5개 기본 스탯 + 파생 스탯 공식 | 🔴 필수 |
| 레벨업 | 경험치 획득, 레벨업 처리 | 🔴 필수 |
| 기본 전투 | 완전 턴제, 명중/회피 판정 | 🔴 필수 |
| 다굴 시스템 | 민첩 ±10% 동시 공격 | 🔴 필수 |
| 크리티컬 | 5% 기본, 방어력 무시 | 🔴 필수 |
| 펫 운용 | 전면 3마리 편성 | 🔴 필수 |
| 탑승 | 캐릭터+펫 합체 유닛 | 🟡 중요 |
| 포획 | 1레벨 펫 포획 (3% 등장) | 🟡 중요 |
| 기본 장비 | 6종 무기 장착 | 🟡 중요 |

### 세부 일정

| 일차 | 작업 내용 |
|------|-----------|
| Day 1-2 | DB 스키마 설계 (캐릭터, 펫, 장비) |
| Day 3-4 | 기본 API 구축 (인증, CRUD) |
| Day 5-6 | 전투 로직 - 턴제 시스템 |
| Day 7-8 | 전투 로직 - 데미지 계산, 명중/회피 |
| Day 9-10 | 펫 시스템 + 포획 |
| Day 11-12 | 다굴 + 크리티컬 |
| Day 13-14 | 기본 전투 UI + 테스트 |

### 완료 조건

- [ ] 캐릭터 1명 + 펫 3마리로 전투 가능
- [ ] 턴제 전투 정상 작동
- [ ] 다굴/크리티컬 발동 확인
- [ ] 펫 포획 가능
- [ ] 경험치 획득/레벨업 작동

## Phase 2: 캐릭터/스테이지 (2주)

### 목표

게임 구조 완성

### 포함 기능

| 기능 | 상세 | 우선순위 |
|------|------|----------|
| 캐릭터 생성 | 외형 랜덤, 닉네임, 속성/스탯 선택 | 🔴 필수 |
| 속성 시스템 | 4속성 상성, 복합 속성 | 🔴 필수 |
| 스테이지 | 순차 해금, 몬스터 배치 | 🔴 필수 |
| 5배수 웨이브 | 다중 웨이브 전투 | 🟡 중요 |
| 10배수 보스 | 보스 전투 | 🟡 중요 |
| 별점 시스템 | 3개 조건 평가 | 🟡 중요 |
| 충성도 | 0~100, 불복/도주 | 🟡 중요 |

### 세부 일정

| 일차 | 작업 내용 |
|------|-----------|
| Day 1-2 | 캐릭터 생성 UI + 로직 |
| Day 3-4 | 속성 시스템 구현 |
| Day 5-6 | 스테이지 구조 + 몬스터 배치 |
| Day 7-8 | 웨이브/보스 전투 |
| Day 9-10 | 별점 시스템 |
| Day 11-12 | 충성도 시스템 |
| Day 13-14 | 통합 테스트 |

### 완료 조건

- [ ] 캐릭터 생성 플로우 완료
- [ ] 속성 상성 데미지 계산 작동
- [ ] 스테이지 1~10 플레이 가능
- [ ] 별점 획득 가능
- [ ] 충성도 변화/불복 작동
